# ارزیابی عملی و تخصصی

## خلاصه

سیستم ارزیابی GraphRAG اکنون به طور ویژه بر عملی بودن و تخصصی بودن محتوا تأکید می‌کند. این ویژگی به مدل GPT-4o دستور می‌دهد تا به متن‌هایی که اطلاعات عملی‌تر، تخصصی‌تر و کاربردی‌تری ارائه می‌دهند امتیاز بالاتری بدهد.

## ویژگی‌های جدید

### 1. نوع مقایسه جدید
- **تمرکز بر عملی بودن و تخصصی بودن**: این گزینه به عنوان گزینه پیش‌فرض انتخاب شده است
- معیارهای ارزیابی شامل:
  - عملی بودن: اطلاعات کاربردی و قابل استفاده
  - تخصصی بودن: استفاده از اصطلاحات تخصصی و مفاهیم دقیق علمی
  - دقت علمی: صحت اطلاعات و استناد به مفاهیم علمی
  - جامعیت: پوشش کامل جنبه‌های مختلف موضوع
  - وضوح: قابل فهم بودن برای مخاطب تخصصی

### 2. پرامپ بهبود یافته
پرامپ ارسال شده به GPT-4o اکنون شامل دستورالعمل‌های خاص برای ارزیابی عملی و تخصصی بودن است:

```
**معیارهای ارزیابی مهم:**
1. **عملی بودن**: متن باید اطلاعات کاربردی و قابل استفاده ارائه دهد
2. **تخصصی بودن**: استفاده از اصطلاحات تخصصی و مفاهیم دقیق علمی
3. **دقت علمی**: صحت اطلاعات و استناد به مفاهیم علمی
4. **جامعیت**: پوشش کامل جنبه‌های مختلف موضوع
5. **وضوح**: قابل فهم بودن برای مخاطب تخصصی
```

### 3. پیام سیستم بهبود یافته
پیام سیستم ارسال شده به GPT-4o اکنون تأکید ویژه‌ای بر عملی بودن و تخصصی بودن دارد:

```
"شما یک متخصص ارزیابی کیفیت متن با تخصص در زمینه‌های علمی و عملی هستید. وظیفه شما مقایسه دو متن با تمرکز ویژه بر عملی بودن، تخصصی بودن، و کاربردی بودن محتوا است. به متن‌هایی که اطلاعات عملی‌تر، تخصصی‌تر و کاربردی‌تری ارائه می‌دهند امتیاز بالاتری بدهید."
```

## نحوه استفاده

### در رابط کاربری
1. به صفحه ارزیابی بروید
2. دو متن مختلف را وارد کنید
3. در بخش "مقایسه هوشمند با GPT"، گزینه "تمرکز بر عملی بودن و تخصصی بودن" را انتخاب کنید
4. دکمه "مقایسه با GPT" را کلیک کنید

### از طریق API
```python
import requests

data = {
    "text1": "متن تخصصی با جزئیات علمی",
    "text2": "متن عمومی و ساده",
    "label1": "روش تخصصی",
    "label2": "روش عمومی",
    "comparison_type": "practical_specialized",
    "gpt_model": "gpt-4o"
}

response = requests.post('http://localhost:5000/api/compare_with_gpt', json=data)
result = response.json()
```

## مثال‌های ارزیابی

### متن تخصصی (امتیاز بالاتر)
```
ژن TP53 یک ژن سرکوب‌کننده تومور است که نقش مهمی در تنظیم چرخه سلولی و آپوپتوز دارد. جهش‌های این ژن در بسیاری از سرطان‌ها مشاهده می‌شود و منجر به از دست رفتن عملکرد طبیعی آن می‌شود.
```

### متن عمومی (امتیاز پایین‌تر)
```
TP53 یک ژن مهم در پیشگیری از سرطان است که پروتئین p53 را کد می‌کند. این پروتئین در پاسخ به آسیب DNA فعال می‌شود.
```
